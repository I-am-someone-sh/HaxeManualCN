我们会使用很多 Haxe 源代码来保持理论与实践之间的桥接。这些代码示例通常是完整的程序，并带有一个 main 函数，它们都可以被编译。不过也有时候只有最关键的部分被展示出来。源代码看起来就像这样：

